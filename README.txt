Insights.Dev

To install the database, go to MYSQL Workbench, create a new schema called "Blog".
Then go to Server > Data Import > and select the database folder from this directory.


To Run the Project:
Open this folder in VS Code > Open two terminals and navigate to the api and client folders in one each.
Then execute this commands one by one in both:
1) npm init -y
2) npm i
3) npm start
